Support
Help, without the runaround

If something’s broken, we fix it. If it’s confusing, we simplify it.

Restore purchases

Open the app → Payment → Restore purchases. If it still does not unlock, check that you are signed into the same Apple ID / Google account used to purchase.

If you’re stuck, message support with your account email and platform (iOS/Android). No screenshots of receipts needed unless requested.

Push notifications

If notifications are silent: check system settings first (Antara → Notifications). On Android, also check battery optimisation for Antara.
